Spécifie une collection de noms de loggueurs et de leurs niveaux de log que l'enregistreur de log collectera. Par exemple, si vous spécifiez hudson.model.Hudson et info, tous les logs enregistrés pour hudson.model.Hudson ou ses descendants avec le niveau info ou supérieur, appraitront dans cet enregistreur de log.

Si vous avez des entrées multiples, une demande de log qui correspond à l'un d'entre eux sera bien enregistrée.

Voyez également ce document pour plus de détails sur le logging en Java et avec Hudson.
