Подчиненный узел должен иметь директорию выделенную для Hudson. Укажите полный путь к этой директории, например '/export/home/hudson'.

Подчиненные узлы не содержат важных данный (чего-то кроме директорий сборки последних собранных проектов), так что вы можете назначить в качестве такой директории любую временную. Единственное негативное последствие - вы можете потерять последнюю версию сборочной директории, если узел выключится.
