Если адреса ваших пользователей могут быть вычислены автоматически добавлением суффикса, тогда укажите этот суффикс. Иначе, оставьте поле пустым. Заметьте, что пользователи в любой момент могут вручную поменять свой адрес в настройках. Например, если значение поля @acme.org, то пользователь mike по умолчанию получит адрес mike@acme.org.
