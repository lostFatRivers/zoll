Spécifie optionnellement l'adresse HTTP de l'installation de Hudson, comme http://yourhost.yourdomain/hudson/. Cela est nécessaire dans certaines configurations parce que Hudson ne peut pas détecter de façon fiable sa propre URL.

Cette valeur est utilisée pour mettre des liens dans les e-mails envoyés par Hudson et interne lorsque Hudson envoie des requêtes à lui-même. Par conséquent, si une mauvaise adresse est indiquée, certaines fonctionnalités de Hudson seront brisés.
