Optional: Geben Sie hier die HTTP-Adresse der Hudson-Instanz an, z.B: http://yourhost.yourdomain/hudson/. Dies ist notwendig, weil Hudson nicht zuverlässig seine eigene URL feststellen kann.

Dieser Wert wird verwendet, um Links in E-Mail-Benachrichtigungen einzufügen, und wenn Hudson sendet Anforderungen an sich selbst. Folglich, wenn eine falsche Adresse eingegeben wurde, werden einige Hudson Funktionen gebrochen werden.
